Tools used: 
Backend:
    - body-parser: ver "^1.18.3",
    - express: ver "^4.16.4",
    - mongoose: ver "^5.4.1",
    - node: ver 8.12.0
Frontend:
    - Jquery, Bootstrap

Installation guide:
    - Node modules folder is not included in this submission
    - after extract the file, open terminal and locate yourself into the root folder of this project
    - make sure you have node.js install
    - type "npm install" to install all dependencies
    - type "npm start" to start the project
